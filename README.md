# sw-tech-eval

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## Other Scripts

### Populate db with mock data

```sh
npm run createdb
```

### Drop products table/data (re-run createdb after)

```sh
npm run wipedb
```

### Run local express server; serve db/basic api

```sh
npm run server
```
